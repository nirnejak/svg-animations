## Available Scripts

**Install Dependencies**

```bash
bun install
```

**Start Development Server**

```bash
bun run dev
```

**Lint all the files**

```bash
bun run lint
```

**Format all files with Prettier**

```bash
bun run format
```

**Check TypeScript issues**

```bash
bun run type-check
```

**Build for Production**

```bash
bun run build
```

**Start Production Server**

```bash
bun start
```
