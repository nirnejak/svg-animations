## Available Scripts

**Install Dependencies**

```bash
npm install
```

**Install Git Hooks(Required for Development)**

```bash
npm run prepare
```

**Start Development Server**

```bash
npm run dev
```

**Lint all the files**

```bash
npm run lint
```

**Format all files with Prettier**

```bash
npm run format
```

**Check TypeScript issues**

```bash
npm run type-check
```

**Build for Production**

```bash
npm run build
```

**Start Production Server**

```bash
npm start
```
